How to Use the Annotation Tool

1. Overview

This tool allows you to listen to two versions of a Text-to-Speech (TTS) audio (Male and Female) and evaluate them based on Naturalness, Intelligibility, and Context. You will also identify specific mistakes in the text.

2. Marking Incorrect Words

Feature: Interactive Text Highlighting

The text displayed at the top is interactive. If you hear a mispronunciation or an error in a specific word:

  • Click on the word in the text box.
  • It will turn Red and Strikethrough.
  • This word is automatically added to the "List of IncorrectWords" box for both forms.
  • Click the word again to undo the selection.

3. The "Perfect" Button ✨

If an audio clip is distinctively good and has no errors, you can use the ✨ Perfect button located above the form.

Clicking this will automatically:

  • Set Naturalness to "Human-like".
  • Set Intelligibility to "Yes".
  • Set Context to "Not Applicable".
  • Clear any selected incorrect words or notes.

Use this to speed up annotation for good samples!

4. Evaluation Fields

  • Naturalness: Does the voice sound robotic or like a real human?
  • Intelligibility: Is the audio clear? Can you understand every word?
  • Context: Does the tone match the sentence (e.g., rising tone for questions)? Select "Not Applicable" for simple statements without specific tonal requirements.
  • Number Mistakes (সংখ্যা): List any numbers that were read incorrectly.
  • Conjunct Mistakes (যুক্তাক্ষর): List any issues with complex characters.

5. Saving & Navigation

  • Overall Preference: You MUST select which version (Male or Female) sounded better, or if they were Equal.
  • Save All & Next: Click this button at the bottom to save evaluations for BOTH files and automatically move to the next pending item.
